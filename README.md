# Dummy Todo Application

A simple full-stack Todo application built with React, Vite, Node.js, and Express.

Testing Jenkins Polling

## Structure
- `client/`: React Frontend
- `server/`: Node.js Backend

## Quick Start

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```
